# ASO and Referral Ops Checklist

## ASO weekly loop
- Track store page conversion (view-to-install) by country and locale.
- Test one icon variant and one first-screenshot variant per cycle.
- Keep title/subtitle keyword map aligned with speaking, trial, and tutor intents.
- Refresh release notes with measurable outcomes (for example: faster booking flow).

## Referral attribution loop
- Monitor `referral_copied` and `referral_invite_copied` events daily.
- Compare invite-copy to trial-request conversion by segment and experiment.
- Review top drop-off between trial request and booking start every week.
- Update referral incentive copy from Growth dashboard findings.

## Rollback criteria
- If store conversion drops >10% week-over-week, revert ASO variant.
- If referral invite conversion drops >15%, revert invite copy variant.
